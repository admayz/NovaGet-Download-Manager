# Extension Icons

Place the following icon files in this directory:

- `icon16.png` - 16x16 pixels
- `icon32.png` - 32x32 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

These icons should represent the Download Manager application and will be displayed in:
- Browser toolbar (16x16, 32x32)
- Extension management page (48x48)
- Chrome Web Store (128x128)

## Design Guidelines

- Use a simple, recognizable download icon (e.g., downward arrow)
- Use the application's brand colors
- Ensure icons are clear at all sizes
- Use transparent backgrounds (PNG format)
